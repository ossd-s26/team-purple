# Installation Instructions
_Please note, this is a temporary mode of installation._
You should first create a [local copy](https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository) of the extension. 

Once this is done, you should navigate to `about:debugging` via Mozilla Firefox. Click the **This Firefox** option -> **Load Temporary Add-on** -> Navigate to the project folder and select the `manifest.json` file
