# Contributing to This Project

## Types of Contributions 
- Reporting a bug
- Fixing a bug  
- Implementing a new feature
- Translating the project to a new language
- Updating the documentation

## How to Contribute 

Please use this workflow below to contribute.

1. Fork this repository by clicking on the fork button on the top right of the repository
2. Clone the fork to your local machine 
    
    ```git clone https://github.com/[YOUR-USERNAME]/team-purple.git```

3. Create a branch for your changes

    ```git checkout -b NAME-OF-YOUR-BRANCH```

4. Make your changes
5. Test your changes on Firefox
6. Add and commit your changes 

    ```git add FILE-YOU-CHANGED```
    ```git commit -m "DESCRIPTION"```

7. Push your changes to your forked repo

    ```git push origin NAME-OF-YOUR-BRANCH```

8. Submit a pull request
    a. Click on the "Compare & pull request" in this repo and write a meaningful description
